# VerseGen

A minimal Next.js App Router project with a landing page to select Bible Version, Book, Chapter, and Verse. Clicking Generate renders a list of dummy verses.

https://versegenerator.vercel.app

## Getting Started

1. Install dependencies

```bash
npm install
```

2. Run the development server

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Notes
- The list of books and versions is a small sample; you can extend it in `app/page.tsx`.
- Dummy verses are generated client-side for demo purposes.
